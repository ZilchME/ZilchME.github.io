在终端中将文本文件内容复制到系统剪贴板

对于不同的系统，需要使用不同工具实现

Linux 系统

在 Linux 上，通常需要使用 xclip 或 xsel 等外部工具。

Arch Linux
sudo pacman -S xclip
Ubuntu/Debian
sudo apt install xclip

使用以下命令将文件内容复制到剪贴板：

xclip -selection clipboard < README.md

macOS 系统

macOS 自带了 pbcopy 命令

pbcopy < README.md

Windows 系统

在 Windows 10 (1803版本) 及更高版本中，可以使用内置的 clip 命令。

clip < README.md

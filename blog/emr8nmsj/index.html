按位与闭包的最大基数

问题描述

给定一个初始长度为 nnn 的数组 aaa。你可以进行任意次（包括零次）如下操作：

  • 选择两个下标 i,ji, ji,j（满足 1≤i,j≤m1 \le i, j \le m1≤i,j≤m，其中 mmm 为当前数组长度）；
  • 计算 v=ai & ajv = a_i \ \&\ a_jv=ai​ & aj​（其中 & 表示按位与运算）；
  • 将结果 vvv 追加到数组末尾，使数组长度增加 1。

求在进行了任意次操作后，数组中最多能出现多少个不同的元素。

输入描述

  • 第一行输入一个整数 TTT（1≤T≤1041 \le T \le 10^41≤T≤104），表示数据组数。
  • 对于每组数据：
    • 第一行输入一个整数 nnn（1≤n≤2×1051 \le n \le 2 \times 10^51≤n≤2×105）。
    • 第二行输入 nnn 个整数 a1,a2,…,ana_1, a_2, \dots, a_na1​,a2​,…,an​（0≤ai≤10230 \le a_i \le 10230≤ai​≤1023）。
  • 保证所有测试数据的 nnn 之和不超过 2×1052 \times 10^52×105。

思路

问题定义

设 S0={a1,a2,…,an}S_0 = \{a_1, a_2, \dots, a_n\}S0​={a1​,a2​,…,an​} 为初始整数集合。定义操作 O(x,y)=x & y\mathcal{O}(x, y) = x \ \&\ yO(x,y)=x & y，其中 &\&& 为按位与运算。 我们需要构建一个序列集合 S0⊆S1⊆S2⊆…S_0 \subseteq S_1 \subseteq S_2 \subseteq \dotsS0​⊆S1​⊆S2​⊆…，其中 Sk+1=Sk∪{x & y∣x,y∈Sk}S_{k+1} = S_k \cup \{x \ \&\ y \mid x, y \in S_k\}Sk+1​=Sk​∪{x & y∣x,y∈Sk​}。 求该序列的极限集合 S∞S_{\infty}S∞​ 的基数（即不同元素的个数），记为 ∣S∞∣|S_{\infty}|∣S∞​∣。

实现思路

值域限制在 10 位二进制数范围内（0≤ai≤210−10 \le a_i \le 2^{10} - 10≤ai​≤210−1），故可以通过数组维护一个固定大小为 1024 的集合

  1. 遍历 S0={a1,a2,…,an}S_0 = \{a_1, a_2, \dots, a_n\}S0​={a1​,a2​,…,an​} 中每一个数 aia_iai​
  2. 将 aia_iai​ 与集合中的每一个数按位相与，将 aia_iai​ 本身以及新生成的数加入集合。
  3. 最终的集合大小为所求结果

上述循环可以保证穷举过所有不重复元素的任意组合，不会出现遗漏

优化点： aia_iai​ 可能由之前的数组合得到过，此时可以跳过

解题代码（AMC模式）

import java.io.*;
import java.util.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        int t = Integer.parseInt(br.readLine());
        StringBuilder sb = new StringBuilder();
        
        boolean[] reachable = new boolean[1024];
        int[] reachableValues = new int[1024];

        while (t-- > 0) {
            int n = Integer.parseInt(br.readLine());
            StringTokenizer st = new StringTokenizer(br.readLine());

            Arrays.fill(reachable, false);
            int numReachable = 0;
            
            for (int i = 0; i < n; i++) {
                int val = Integer.parseInt(st.nextToken());

                if (reachable[val]) continue;
                
                // 将当前值与已经发现的所有可达值进行按位与
                int currentLimit = numReachable;
                for (int j = 0; j < currentLimit; j++) {
                    int next = reachableValues[j] & val;
                    if (!reachable[next]) {
                        reachable[next] = true;
                        reachableValues[numReachable++] = next;
                    }
                }
                reachable[val] = true;
                reachableValues[numReachable++] = val;
            }
            sb.append(numReachable).append("\n");
        }
        System.out.print(sb);
    }
}

  • 空间复杂度：O(V)O(V)O(V)，用于存储数据以及记录数值是否出现。
